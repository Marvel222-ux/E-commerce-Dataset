# E-commerce-Dataset

## Tools Used
- Microsoft Excel/Metabase
- Power Bi
- Flaticons
- Pivot Tables
- Charts & Graphs
- Microsoft Power Point
- Data Cleaning Techniques

## Dataset Description
The dataset contains:
- User ID
- Product ID
- Category
- Price
- Discount
- Final price
- Payment_ Method
- Purchase_Date

## Data Cleaning Process
- Removed Duplicate
- Handled missing values
- Standardized data format
- Oranized data for analysis

## PowerPoint Summary Process
- Title
- Business Problem
- Dataset Overview
- KPI Summary
- Dashboard overview
- Key Insight
- Conclusion

## Dashboard Features
- Sum of final_price by category
- Sum of final_price by discount
- Count of category by payment method
- Sum of discount by category

## Key Insights
- Category (Clothing) generated highest Sales
- Credit Card was most used in payment method
- Category (Home & Kitchen) used the highest discount
